# Minishell

A minimalist Unix shell implemented in C. Developed at ENSEEIHT as part of the Operating Systems course.

## 🌐 Languages
- 🇫🇷 French version: `minishell_fr.c` + `Rapport.pdf`
- 🇬🇧 English version: `minishell_eng.c` + `Report.pdf`

## 📁 Structure

├── src/ # Source code
├── reports/ # Project reports (PDF)
├── Makefile # Build automation
└── README.md

## 🧪 How to Use

```bash
make               # Builds both minishell_eng and minishell_fr
./minishell_eng    # Runs the English version
./minishell_fr     # Runs the French version
